# Session Start

Read SESSION.md in the project root. Then:

1. Brief the Director with:
   - **Goal / Objective:** the current mission from the Mission section
   - **In progress:** any tasks with status `in_progress` from the Tasks table
   - **Next steps:** top 3 items from Next Steps
   - **Blockers:** any unresolved blockers

2. Check `.claude/tasks/` for any task cards with status `in_progress` — flag any that appear stale (last updated >24h ago).

3. Ask: "Continue from here, or something new?"

Keep the brief to 5–8 lines. No padding.
