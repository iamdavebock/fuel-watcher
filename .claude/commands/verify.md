# Verify

Review and verify all work completed so far in this session.

Invoke the Reviewer agent with:
- All files changed or created this session
- What each change was meant to accomplish
- The original success criteria (from the plan or Director brief)

The Reviewer checks:
- Did the implementation match the intent?
- Logic errors, edge cases, or regressions introduced?
- Best practices followed? No shortcuts or tech debt?
- Any issues introduced that weren't present before?

Report findings to the Director:
- What was verified
- Issues found (critical / minor)
- Whether the work is solid or needs a follow-up pass
